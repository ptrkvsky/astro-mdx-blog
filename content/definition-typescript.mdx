TypeScript est **un surensemble de JavaScript typé** (facultatif) qui peut aider à construire et à gérer des projets JavaScript à grande échelle. Il peut être considéré comme du JavaScript avec des fonctionnalités supplémentaires comme le **typage statique fort**, la **compilation** et la programmation orientée objet.

<Image alt="Logo Typescript" src={javascriptLogo} />

## Est-ce que le Typescript est différent de javascript ?

TypeScript est un langage de programmation orienté objet développé par Microsoft Corporation, tandis que JavaScript est le langage de programmation pour le web. TypeScript est un langage open-source permettant de créer des applications web à grande échelle, tandis que JavaScript est un langage de programmation côté serveur qui permet de développer des pages web interactives.
