## Définition React JSX ?

En React le JSX c'est :

- Le JSX c'est l'abréviation de JavaScript XML.
- JSX permet d'écrire du HTML dans React.
- JSX facilite l'écriture et l'ajout de HTML dans React.

## Coder avec du JSX

- JSX nous permet d'écrire des éléments HTML en JavaScript et de les placer dans le DOM sans avoir recours aux méthodes createElement() et/ou appendChild().
- JSX convertit les balises HTML en éléments réactifs.

## Exemple

```jsx
const myElement = <h1>I Love JSX!</h1>;

const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(myElement);
```
